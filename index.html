sumit403

A small utility for Jupyter Notebook and Google Colab that injects ready-to-run deep learning experiment templates into the next notebook cell.

Installation

pip install sumit403 -v

Usage

import sumit403

sumit403.p1()

This will inject the code for experiment 1 into a new notebook cell.

Experiments

  • p1() — Feed Forward Neural Network (MNIST)
  • p2() — XOR Perceptron
  • p3() — Image Classification Model
  • p4() — Simple CNN
  • p5() — Sentiment Analysis (RNN)
  • p6() — LSTM Autoencoder
  • p7() — GAN Image Generation
