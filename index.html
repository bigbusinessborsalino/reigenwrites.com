sumit403

A Jupyter / Google Colab snippet injector that inserts ready-to-run deep-learning experiment templates into the next notebook cell. Designed for students and learners to quickly prototype experiments inside notebooks.

Quick start

pip install sumit403

# in a Jupyter cell:
import sumit403
sumit403.p1()  # inserts experiment 1 into the next cell

Available experiments (short)

  • p1() — Feed-forward MNIST example (Flatten → Dense → Softmax)
  • p2() — XOR perceptron demo (tiny MLP)
  • p3() — Image classification training flow (train / val)
  • p4() — Small CNN demo (PyTorch)
  • p5() — Sentiment analysis (RNN / LSTM, IMDB)
  • p6() — LSTM autoencoder for time-series reconstruction
  • p7() — Small GAN training loop (generator + discriminator)

Google Colab usage

!pip install sumit403
import sumit403
sumit403.p3()  # inserts the image-classification template

Notes & tips

  • Snippets contain heavy-library imports inside the injected code (e.g., tensorflow, torch). Those libraries are not imported at package import time.
  • Install only the libraries you need for the experiment you plan to run, e.g. pip install tensorflow matplotlib.
  • For heavy experiments (GANs / large LSTMs) use a fresh Colab runtime and enable GPU.

Ethics & academic integrity

These templates are educational. Do not use them to submit others’ work or bypass academic policies. Use them to learn, prototype, and test ideas ethically.
