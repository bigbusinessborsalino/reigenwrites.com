sumit403

A small utility for Jupyter Notebook and Google Colab that injects ready-to-run deep learning experiment templates into the next notebook cell.

Note: The library automatically checks for and installs all missing dependencies (like Torch or Matplotlib) when an experiment is called.

Installation

pip install sumit403==0.4

Usage

import sumit403

sumit403.p1()

This will inject the code for experiment 1 into a new notebook cell.

Special Setup for p4() (PyTorch)

Experiment p4() requires a dedicated environment. The library will handle the heavy lifting for you.

After running sumit403.p4(), you will receive a pop-up notification once the new environment is successfully created.

Step 1: Restart the Kernel

Go to the top menu in Jupyter, click on Kernel, and select Restart Kernel...

Step 2: Select the Environment

In the "Select Kernel" dialog, choose either "Python (403-Lab)" or "Python [conda env:403-env]" to start the experiment with the correct dependencies.

Experiments

  • p1() — Feed Forward Neural Network (MNIST)
  • p2() — XOR Perceptron
  • p3() — Image Classification Model
  • p4() — Simple CNN (Requires 403-Lab Kernel)
  • p5() — Sentiment Analysis (RNN)
  • p6() — LSTM Autoencoder
  • p7() — GAN Image Generation
